# Wall Chess PWA Game

تطبيق لعبة Wall Chess بتقنية Progressive Web App (PWA) مطور بـ React.

## الميزات

- ✅ لعبة Wall Chess كاملة للاعبين
- ✅ اللعب ضد الكمبيوتر بثلاث مستويات صعوبة (سهل، متوسط، صعب)
- ✅ واجهة مستخدم عربية
- ✅ تقنية PWA للتثبيت على الهاتف
- ✅ تصميم متجاوب للهواتف والحاسوب
- ✅ Service Worker للعمل بدون إنترنت
- ✅ 8 جدران لكل لاعب
- ✅ إمكانية القفز فوق اللاعب الآخر
- ✅ زر التراجع للعودة خطوة واحدة
- ✅ مؤثرات صوتية تفاعلية
- ✅ رسالة ترحيب مع شرح قواعد اللعبة
- ✅ تصميم داكن جذاب

## قواعد اللعبة

1. **الهدف**: كن أول لاعب يصل بقطعته إلى الجانب الآخر من اللوحة
2. **الحركة**: يمكن تحريك القطعة مربع واحد في أي اتجاه (أعلى، أسفل، يمين، يسار)
3. **القفز**: إذا كان هناك لاعب آخر في المربع المجاور، يمكن القفز فوقه
4. **الجدران**: يمكن وضع جدار لمنع حركة الخصم (كل جدار يغطي مربعين)
5. **القيود**: لا يمكن وضع جدار يمنع اللاعب من الوصول إلى هدفه نهائياً

## كيفية اللعب

1. اللاعب الأحمر (♜) يبدأ من الأسفل ويحاول الوصول للأعلى
2. اللاعب الأزرق (♞) يبدأ من الأعلى ويحاول الوصول للأسفل
3. في كل دور، يمكن للاعب إما:
   - تحريك قطعته
   - وضع جدار (إذا كان لديه جدران متبقية)
4. استخدم زر "تبديل الوضع" للتنقل بين وضع التحريك ووضع وضع الجدران
5. يمكن استخدام زر "تراجع" للعودة خطوة واحدة

## مستويات الذكاء الاصطناعي

- **سهل 🤖**: حركات عشوائية
- **متوسط 🎯**: يحاول الوصول للهدف مع بعض العشوائية
- **صعب 🧠**: استراتيجية متقدمة تحاول تقليل مساره وزيادة مسار الخصم

## التطوير

```bash
cd wall-chess
pnpm install
pnpm run dev
```

## البناء للإنتاج

```bash
pnpm run build
```

## التقنيات المستخدمة

- React 18
- Vite
- Tailwind CSS
- PWA (Service Worker + Web App Manifest)
- shadcn/ui components
- Web Audio API للمؤثرات الصوتية

## هيكل المشروع

- `src/App.jsx` - منطق اللعبة الرئيسي والذكاء الاصطناعي
- `src/App.css` - تصميم اللعبة
- `public/manifest.json` - إعدادات PWA
- `public/sw.js` - Service Worker
- `public/icon-*.png` - أيقونات التطبيق
